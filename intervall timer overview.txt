Project Overview — Interval Timer App (Mobile, Local-First)
Project Summary

Eine lokale, minimalistische Intervall-Timer-App für Android, die Nutzer regelmäßig per Sound, Vibration und Notification erinnert (z. B. Trinken, Pausen, Fokus).
Die App ist accountlos, offline, extrem simpel, und folgt einer cleanen, Teenage-Engineering-inspirierten Designsprache.

Kernwert: Zuverlässige, konfigurierbare Erinnerungen mit minimalem UI-Overhead.

Goals & Non-Goals
Goals

Zuverlässige Intervall-Erinnerungen im Background + Lockscreen

Extrem schnelle Bedienung mit einer Hand

Klare visuelle Trennung der Timer (Farben)

Vollständig lokal, kein Account, kein Sync

Non-Goals

❌ Kein Cloud-Sync

❌ Kein iOS (vorerst)

❌ Keine Analytics / Telemetrie

❌ Keine Accounts / Profile

❌ Keine komplexen Wizards oder Beispiel-Timer

Target Users & Roles

Rolle: Single User (lokal)

Zielgruppe (breit, bewusst unspezifisch):

Menschen, die regelmäßige Erinnerungen brauchen

Fokus- / Routine- / Health-Use-Cases

Technikaffine Nutzer, die keinen Overhead wollen

Core Use Cases / User Stories

Als User öffne ich die App und sehe alle Timer in einer Liste

Ich kann mindestens 8 Timer gleichzeitig sehen

Ich erstelle einen neuen Timer über einen zentralen Plus-Button

Ich konfiguriere Intervall, Name, Farbe, Sound, Vibration

Ich starte/stoppe jeden Timer direkt in der Liste

Ein Timer erinnert mich alle X Minuten zuverlässig

Die Erinnerung funktioniert auch bei Screen off / App im Hintergrund

Ich kann Timer per Swipe nach links löschen

Ich kann Ruhezeiten definieren (z. B. nachts)

Ich kann eigene Audio-Dateien auswählen

Ich kann einen globalen Mute-Button aktivieren

Die App startet im Dark Mode

Ich kann die Sprache auf DE / EN umstellen
